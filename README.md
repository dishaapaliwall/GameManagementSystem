# Game Management System 

A robust desktop application built with C# Windows Forms and MySQL to manage a comprehensive gaming ecosystem. The system features role-based access control, allowing Admins, Developers, and Players to interact with the platform securely and efficiently.

##  Key Features

###  Role-Based Dashboards
- **Player Dashboard**: Browse available games, view player statistics, purchase games, and launch installed games.
- **Developer Dashboard**: Publish new games, track game sales, view buyer analytics, and manage published content.
- **Admin Dashboard**: Manage user accounts, verify and approve published games, oversee platform operations, and monitor system-wide analytics.

###  Advanced Database Integration
- **Relational Schema**: Fully normalized database schema with complex One-to-Many and Many-to-Many relationships.
- **Transactions (ACID)**: Secure game purchasing and wallet updates using database transactions to ensure data integrity.
- **Stored Procedures**: Encapsulated business logic for complex queries, enhancing security and performance.
- **Views & Indexing**: Optimized data retrieval for fast and responsive dashboards.

##  Technologies Used

- **Frontend**: C# (Windows Forms / .NET)
- **Backend / Database**: MySQL
- **IDE**: Visual Studio
- **Architecture**: Client-Server Model

##  Prerequisites
